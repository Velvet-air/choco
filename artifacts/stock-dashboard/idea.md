# 주식 정보 조회 대시보드

## Problem Statement
HMW — 내가 관심 있는 종목의 핵심 정보를, 인증 없이 종목명만 입력하는 것만으로 한눈에 파악할 수 있게 할까?

## Recommended Direction
"종목 검색 → 즉시 정보 표시" 단일 플로우.

종목명이나 티커를 입력하면 현재가·등락률·금일 고저가·차트·뉴스 헤드라인을 한 화면에 보여준다.
`yahoo-finance2` 패키지(인증 불필요)를 사용하면 한국(005930.KS)·미국 주식 모두 커버 가능하다.
워치리스트는 로컬스토리지에 저장해 서버 없이 유지한다.

## Key Assumptions to Validate
- [ ] yahoo-finance2로 한국 주식 데이터 조회 성공 여부 (005930.KS 호출 테스트)
- [ ] 뉴스 헤드라인이 한국 종목에도 충분히 제공되는가 (일부 종목은 영문 뉴스만 있을 수 있음)
- [ ] 차트용 일봉 히스토리 데이터가 무료로 안정적으로 제공되는가

## MVP Scope
포함:
- 종목명/티커 검색 입력
- 현재가, 등락률 (금액 + 퍼센트)
- 금일 최저가 / 최고가
- 일봉 차트 (최근 1개월)
- 뉴스 헤드라인 목록 (제목 + 링크)
- 워치리스트 (로컬스토리지 저장)

빠짐:
- 재무제표, PER/PBR 등 펀더멘털 지표
- 조건 달성 알림 (push notification)
- 계좌 연동 / 보유 손익 계산
- 소셜·공유 기능

## Not Doing (and Why)
- API 인증·유료 플랜 — 단순성 유지, 무료 범위로 충분
- 실시간 틱 스트리밍 — Yahoo Finance 15분 지연, 개인 조회엔 충분
- 차트 커스터마이징 (지표·보조선 등) — MVP 이후 필요 시 추가
- 모바일 네이티브 앱 — 반응형 웹으로 충분

## Open Questions

- 차트 기간 우선순위: 당일 분봉 vs 최근 1개월 일봉?
- 워치리스트 유지 필요 여부, 아니면 매번 검색해도 충분한가?
