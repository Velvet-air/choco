## 개요

사용자가 관심 있는 한국 주식 종목을 검색하여 워치리스트에 추가하면,
모든 종목의 현재가·등락률·고저가·가격 방향·뉴스 헤드라인이 한 화면에 가로로 나열된다.

## 범위

### 포함
- 종목 검색 및 워치리스트 추가/제거
- 종목별 컬럼: 현재가, 등락률(금액·퍼센트), 금일 고저가, 스파크라인, 뉴스 헤드라인(제목+링크)
- 워치리스트 브라우저 유지 (새로고침 후에도 복원)
- 한국 주식 전용

### 제외
- 미국 주식 지원 — MVP 이후 확장
- 차트 상세(기간 선택, 보조 지표) — 방향 파악만으로 충분
- 실시간 자동 갱신 — 무료 API 제약
- 재무제표·PER/PBR 등 펀더멘털 — MVP 범위 외
- 조건 달성 알림 — MVP 이후
- 계좌 연동·보유 손익 계산 — MVP 범위 외

## 시나리오

### 1. 워치리스트가 비어 있을 때

- **Given** — 처음 접속하거나 모든 종목을 제거한 상태
- **When** — 페이지를 열면
- **Then** — "종목을 추가하세요" 안내 문구와 검색 입력창이 표시된다

성공 기준:
- [ ] 워치리스트가 비어 있을 때 → "종목을 추가하세요" 문구가 화면에 보인다
- [ ] 검색 입력창이 비어 있고 포커스 가능한 상태로 표시된다

### 2. 종목 검색 및 추가

- **Given** — 검색 입력창이 표시된 상태
- **When** — 종목명 또는 티커를 입력하고 추가를 확인하면
- **Then** — 해당 종목 컬럼이 워치리스트 맨 뒤에 추가되고 데이터가 표시된다

성공 기준:
- [ ] 유효한 티커 입력 → 종목 컬럼이 나타나고 종목명이 표시된다
- [ ] 추가된 종목은 페이지 새로고침 후에도 워치리스트에 유지된다
- [ ] 이미 추가된 종목을 다시 추가하면 중복 컬럼 없이 기존 컬럼이 유지된다

### 3. 종목 정보 표시

- **Given** — 워치리스트에 종목이 1개 이상 있는 상태
- **When** — 페이지가 로드되면
- **Then** — 각 종목 컬럼에 현재가·등락률·금일 고저가·스파크라인·뉴스 헤드라인이 가로로 나열되어 표시된다

성공 기준:
- [ ] 현재가가 원화로 표시된다 (예: 75,400원)
- [ ] 등락률이 금액과 퍼센트로 함께 표시된다 (예: +900원 / +1.2%)
- [ ] 상승 시 등락률이 빨간색, 하락 시 파란색으로 표시된다
- [ ] 금일 최고가와 최저가가 각각 표시된다
- [ ] 최근 가격 방향을 나타내는 스파크라인(미니 라인)이 표시된다
- [ ] 뉴스 헤드라인이 최대 5개 표시되며(6개 이상이면 5개만 노출), 클릭 시 새 탭으로 원문 링크가 열린다

### 4. 종목 선택 및 제거

- **Given** — 워치리스트에 종목이 있고, 아무 종목도 선택되지 않은 상태
- **When** — 종목 컬럼을 클릭하면
- **Then** — 해당 컬럼이 선택 상태로 표시되고, 상단 − 버튼이 활성화된다

- **Given** — 종목 컬럼이 선택된 상태
- **When** — 상단 − 버튼을 클릭하면
- **Then** — 해당 종목 컬럼이 즉시 사라지고, 새로고침 후에도 복원되지 않는다

성공 기준:
- [ ] 종목 컬럼 클릭 → 해당 컬럼에 선택 표시(강조 테두리)가 나타난다
- [ ] 선택된 종목 없을 때 → 상단 − 버튼이 비활성(disabled) 상태로 표시된다
- [ ] 종목 선택 후 − 버튼 클릭 → 해당 종목 컬럼이 화면에서 사라진다
- [ ] 새로고침 후 제거된 종목이 다시 나타나지 않는다

### 5. 유효하지 않은 종목 입력

- **Given** — 검색 입력창에 존재하지 않는 종목명/티커를 입력한 상태
- **When** — 추가를 확인하면
- **Then** — "종목을 찾을 수 없습니다" 안내가 표시되고, 워치리스트는 변경되지 않는다

성공 기준:
- [ ] 존재하지 않는 티커 입력 → "종목을 찾을 수 없습니다" 메시지가 표시된다
- [ ] 기존 워치리스트 종목 컬럼은 그대로 유지된다

### 6. 데이터 조회 실패

- **Given** — 종목이 워치리스트에 있는 상태
- **When** — 네트워크 오류 등으로 데이터를 가져오지 못하면
- **Then** — 해당 종목 컬럼에 "데이터를 불러올 수 없습니다" 메시지가 표시된다

성공 기준:
- [ ] 조회 실패한 종목 컬럼 → "데이터를 불러올 수 없습니다" 메시지가 표시된다
- [ ] 다른 종목 컬럼의 데이터는 정상적으로 표시된다

## 불변 규칙

- **등락률 색상**: 상승은 빨간색, 하락은 파란색 (한국 주식 관행)
- **가격 단위**: 원화(원) 표시, 소수점 없음

## 의존성

- 한국 주식 시세·뉴스 데이터를 제공하는 외부 데이터 소스

## 미결정 항목

- 스파크라인 기준 기간: 최근 5거래일 vs 오늘 당일 (데이터 가용성에 따라 plan 단계에서 결정)
- 뉴스가 제공되지 않는 종목의 처리: 뉴스 영역 숨김 vs "뉴스 없음" 표시
